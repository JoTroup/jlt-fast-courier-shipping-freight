== Description ==

Fast Courier is an Australian Courier & Freight shipping platform. Connect your WooCommerce Store with a network of Courier & Freight Providers. 
Quickly compare courier quotes and send packages Australia-wide using our WooCommerce comparison tool.  We have built cutting edge logistics technology to help you compare courier and freight quotes, multiple carriers and delivery times. You'll be sure to find the right carrier to match your budget, plans and logistics preferences.
Simply install the Fast Courier Plugin and set-up to retrieve shipping quotes for each and every eCommerce transaction and get your package moving.

For backwards compatibility, if this section is missing, the full length of the short description will be used, and
Markdown parsed.

A few notes about the sections above:

* Connects to the Fast Courier Customer Portal
* Easily Manage all Orders regardless of Courier Carrier
* Wide Selection of Couriers & Shippers
* Create shipping labels for all carriers in one place
* Automatic shipping label creation
* Automatic pickup locations update
* Provide shipping insurance
* Custom Shipping pricing for Products and Cart Value

Automate Everything. Book, Print and Ship.

We’ve negotiated rates with Australia’s best courier providers so you don’t have to. No account keeping fees, no credit checks, no lock-in contracts.
Get your products to customers fast from a warehouse, a store or both.
See more about Fast Courier Services here: [https://fastcourier.com.au](https://fastcourier.com.au)

== Dependencies ==
1. Uses "dvdoug/boxpacker" to calculate packages.
2. Calls FastCorier APIs, to verify merchant and quote selection.
3. Uses external libraries like > SweetAlert, Select2, FontAwesome, BootStrap.

== Frequently Asked Questions ==

= What's the logic behind quote selection? =

This plugin will return fastest and the cheapest quote based of destination and pickup location.
